Demos for jquery.select2autocomplete


Basic example

In the basic example, the default number of suggestions will be made (10).

$('#simple').select2autocomplete();

max_suggestions option

The maximum number of suggestions the widget will pull back is determined by the Max_hits option when setting up the widget. The example has lots of US cities beginning with A

$('#max-hits').select2autocomplete({max_suggestions: 5});

initially selected

The initial value of the select is used to populate the widget's inital value

$('#initial').select2autocomplete();

Tabindex

The tabindex of the original select is copied on to the created input element. If you inspect the generated input you'll see that the tabindex of 5 has been moved on to the input element

$('#tabindex').select2autocomplete();

Focus example

In the basic example, the default number of suggestions will be made (10).

$('#get-focus').select2autocomplete();
